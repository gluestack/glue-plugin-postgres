# CLI Reference

This command will display a list of available commands for the **postgres instance**.
To execute this command using your **Gluestack App**, please run the following command:

## node glue postgres:list

To display the list of **postgres instances** that have been installed, you can use the following command:
```sh
$ node glue postgres:list
```

## node glue postgres:config

To modify the **configurations** of your **postgres instance**, you can use the following command:

```sh
$ node glue postgres:config
```

This command will display a list of available instances and you can select one.
After that you will be prompted for the configuration.
